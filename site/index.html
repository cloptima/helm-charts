Cloptima Helm Charts

This GitHub Pages site hosts the Helm repository for the Cloptima Kubernetes Agent and any future public charts.

Use the repository

Add the repo to Helm and install the agent:

helm repo add cloptima https://cloptima.github.io/helm-charts
helm repo update
helm install cloptima-k8s-agent cloptima/cloptima-k8s-agent \
  --namespace cloptima --create-namespace \
  --set config.customerID="<your-customer-uuid>" \
  --set config.cloudAccountID="<your-cloud-account-uuid>"

The Helm index.yaml backing this repo is available directly at https://cloptima.github.io/helm-charts/index.yaml.
